# 项目简介

本项目是一个Flutter学习项目，以Android开发者的视角展开。项目绝大部分内容来自[Flutter官方网站](https://flutter.cn)，还有一部分来自个人Flutter实践。  
本项目包含了进行Flutter开发所需的基本技能，主要有：

* Flutter的基本概念
* 页面跳转
* 动画
* 常用的布局
* ListView和GridView
* Flutter常用控件
* 网络操作
* 数据库操作
* SharedPreferences使用
* WebView使用
* 第三方应用唤起
* 自定义组件
* 自绘组件
